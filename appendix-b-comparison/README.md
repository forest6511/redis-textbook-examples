# appendix-b-comparison — Redis / Valkey / Dragonfly / Memcached 比較

『Redis の教科書』付録B は、Redis と関連プロダクト（Valkey / Dragonfly / Memcached）を 5 観点（データ型・永続化・実行モデル・ライセンス・互換性）で比較した解説です。実行用のサンプルコードはありません。

各プロダクトの一次情報は以下を参照してください。

- Redis: https://redis.io/docs/
- Valkey: https://valkey.io/
- Dragonfly: https://www.dragonflydb.io/
